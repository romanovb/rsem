Описание данных

В данной работе используем индекс промышленного производства для пищевой промышленности.

Загрузим данные.

food <- sophisthse('PISH_C')
head(food)
##          PISH_C_SA PISH_C
## Jan 1993     100.0   84.0
## Feb 1993     100.3   89.8
## Mar 1993      99.1  100.6
## Apr 1993      97.8   94.8
## May 1993      96.8   92.2
## Jun 1993      95.9  101.9

Нас интересует исходный ряд, поэтому избавимся от столбца с уже скорректированными значениями.

food <- food[,-1]
head(food)
##        Jan   Feb   Mar   Apr   May   Jun
## 1993  84.0  89.8 100.6  94.8  92.2 101.9

Построим простой график нашего временного ряда.

food %>%
  autoplot()

Для повышения корректности модели оставим данные с 2000 года.

foods <- food %>%
  window(start = 2000)

Построим новый график.

foods %>%
  autoplot()

Исследование закономерных компонентов

Рассмотрим закономерные компоненты этого временного ряда отдельно друг от друга.

Уровень

foodmean <- mean(foods)

  autoplot(cbind('Ряд' = foods, 'Уровень' = foodmean)) +
    labs(title = "Уровень ряда",
         x = "Время",
         y = "Значение",
         color = "Легенда") +
    geom_label(aes(x=2005.1, y = 100), label = round(foodmean,3), inherit.aes = FALSE, color = "lightblue" )

В рассматриваемом периоде среднее значение ряда составляет 105,741 единиц.

Тренд

  autoplot(foods, series = "Исходный ряд") +
  autolayer(ma(foods, order = 13), series = "Тренд") +
    labs(title = "Тренд",
         x = "Время",
         y = "Значение",
         color = "Ряды") +
  scale_color_manual(values = c("blue", "red"))

Прослеживается тренд на увеличение индекса производства.

Сезонность

 autoplot(foods)+

         x = "Время",
         y = "Значение")

food11 <- food %>%
 window(start = 2000, end = 2002) 
  autoplot(food11)

Наблюдаются резкие спады с периодичностью 12 месяцев, а также повторяющиеся колебания в течение года.

Декомпозиция временного ряда и модель

Используем мультипликативную модель, поскольку со временем колебания усиливаются.

foodd <- foods %>%
  decompose(type = "multiplicative")
foods1 <- foods %>% stl(s.window = 7, t.window = 21)
foodadj <- seasadj(foods1)

autoplot(foods, series = "Исходный ряд") +
  autolayer(trendcycle(foodd), series = "Тренд") +
  autolayer(foodadj, series = "Скорректированный ряд") +
  scale_color_manual(values = c('red', 'blue', 'green')) +
  labs(title = "Декомпозиция временного ряда",
       color = "Легенда")

Остаток ряда

autoplot(remainder(foodd)) + labs(
  title = "Остаток ряда",
  y = "Значение", 
  x = "Время"
)

mean(remainder(foodd), na.rm = TRUE)
## [1] 0.9997292

Выводы

Характер поведения

Тренд : Увеличение индекса
Сезонность : присутствуют сезонные колебания в начале года и незначительные в течение года
Остаток : среднее значение по остатку близко к нулю

Причины

Тренд на увеличение объемов производства можно объяснить общим ростом российской экономики. Сезонность в начале года может быть обусловлена необходимостью остановки на новогодние праздники даже у круглогодичных производств. Остаток носит случайный характер.
